# 🍽️ OrderAI - Intelligent Restaurant Assistant
### TRY IT NOW AT : https://www.chokbar.fr/
## 📌 Overview
**OrderAI** is an AI-powered restaurant assistant that helps customers browse the menu, place orders, and get real-time order updates via a user-friendly interactive interface. It integrates OpenAI's GPT-based model to handle customer queries and manage orders seamlessly.

## 🚀 Features
- 📝 **Menu Inquiry**: Ask about ingredients, prices, and details of menu items.
- 🛒 **Order Management**: Add or remove items from your order with special instructions.
- 🔄 **Real-Time Order Display**: View current orders in a graphical user interface (GUI).
- 📦 **Order Finalization**: Generate and store order details with unique order numbers.
- 🔑 **Secure API Key Handling**: Uses `.env` for secure storage of API keys.

## 🏗️ Installation

### **1. Clone the Repository**
```bash
git clone https://github.com/cedricdsst/orderAI.git
cd orderAI
```

### **2. Set Up a Virtual Environment (Optional but Recommended)**
```bash
python -m venv venv
source venv/bin/activate  # On Windows use: venv\Scripts\activate
```

### **3. Install Dependencies**
```bash
pip install -r requirements.txt
```

### **4. Create a `.env` File for API Key**
Create a `.env` file in the root folder and add your OpenAI API key:
```
OPENAI_API_KEY=your_openai_api_key_here
```

## 🏃 Usage
Run the application with:
```bash
python app.py
```
- The interactive AI chat will handle customer inquiries and orders.
